## 伪类与伪元素的区别

它的功能和class有些类似，但它是基于文档之外的抽象，所以叫伪类。

与伪类针对特殊状态的元素不同的是，伪元素是**对元素中的特定内容**进行操作

## 伪元素

```css
::after
::before
::first-letter
::first-line
::selection
```
