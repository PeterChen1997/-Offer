## 单体模式

将一批相关的属性和方法组织在一起

优点：将代码封装，避免污染全局作用域

```js
var Singleton = {
    attribute:true,
    method1:function(){},
　　 method2:function(){}
}
```

## 工厂模式

根据调用者的参数生成对象

优点：适合依赖具体环境创建不同实例的场景
