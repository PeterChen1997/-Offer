# Linux常用指令

## pwd = print working directory

当前工作目录

## sudo = superuser do

以管理员权限运行

## reboot = restart computer

重启

## ls = list

显示目标列表

## mv = move

移动文件

## ping = package internet grouper

测试主机之间网络的连通性

## ssh = secure shell

ssh命令是openssh套件中的客户端连接工具，可以给予ssh加密协议实现安全的远程登录服务器

## mkdir = make directory

新建文件夹

## rm -rf file

删除file文件夹及其内容

## find / -name file1

从 '/' 开始搜索文件和目录中有 file1 的文件
