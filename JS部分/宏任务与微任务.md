# 宏任务与微任务

## 具体内容

### macrotasks

- script(整体代码), setImmediate, setTimeout, setInterval, I/O, UI rendering

### microtasks

- process.nextTick, Promises, Object.observe, MutationObserver

## 例子

```js
setImmediate(function () {
  console.log(1);
}, 0);
setTimeout(function () {
  console.log(2);
}, 0);
new Promise(function (resolve) {
  console.log(3);
  resolve();
  console.log(4);
}).then(function () {
  console.log(5);
});
console.log(6);
process.nextTick(function () {
  console.log(7);
});
console.log(8);
// 3 4 6 8 7 5 1 2
```
